# 商店与货架

货架是当前交易闭环的主要可见入口。磨坊、面包店、酒馆和杂货相关内容会逐步通过货架参与交易。

## 当前货架类型

源码中已有这些货架场景：

| 货架 | 用途 |
|---|---|
| flour_shelf | 面粉货架 |
| bread_shelf | 面包货架 |
| grocery_shelf | 杂货货架 |
| tavern_bar_shelf | 酒馆吧台货架 |

## 当前交易工具

NPC 和后台上下文中已经有这些货架相关动作：

| 工具 | 作用 |
|---|---|
| view_shelf | 查看货架 |
| update_shelf | 更新货架库存 |
| buy_from_shelf | 从货架购买 |

## 货架定位

- 货架不是凭空刷货点。
- 库存应来自 NPC 的生产、采购或搬运。
- 面包店和磨坊已经是最明确的早期交易入口。
- 经济闭环的目标是让资源短缺、价格和库存来自真实供需。
