# NPC

NPC 是小镇生活体和经济循环的一部分。他们会被感知、会说话、会被自然语言指令引用，并逐步接入农场、地点、货架和后台 agent 上下文。

## 当前口径

| 来源 | 数量口径 |
|---|---|
| `game-mechanics.md` 当前可玩机制 | 25 名 NPC |
| `town-npcs.md` 花名册和城镇数据 | 约 50 名 NPC |

写当前玩法攻略时以可玩机制为准。写世界设定、职业和经济时可参考完整花名册。

## 主要群体

| 群体 | 代表地点 | 分工 |
|---|---|---|
| 农户和农工 | 北墙麦圃、灰石农圃、圣钟草药园 | 种植、浇水、收获、种子和粮食 |
| 教堂和医护 | 圣烛教堂、圣钟教堂 | 教堂事务、医疗、草药照护 |
| 矿工 | 铁矿、银矿、金矿 | 矿石产出 |
| 木材和燃料链 | 伐木场、炭窑 | 木材、木杆、木炭 |
| 铁匠链 | 锻造场、铁匠铺 | 冶炼、打部件、组装工具 |
| 面粉和面包链 | 磨坊、港口磨坊、面包店 | 小麦、面粉、面团、面包 |
| 畜牧和食物 | 畜牧场、肉铺、酒馆 | 生肉、蛋、熟食、热食 |
| 商业和服务 | 集市、杂货店、裁缝、客栈 | 日用品、纤维、绳、进口货、住宿 |
| 守卫和王室 | 守卫岗、训练场、城门、监狱、王室采办 | 治安、工资、采购 |

## 当前 NPC 的定位

这一版里 NPC 更适合理解为：

- 世界里可见、可感知、可说话的角色。
- 后台 agent 系统的实体入口。
- 行业、货架、地点和经济设计的承载者。
- 还不是完整开放式任务树 NPC。

## 农户默认物品

农户组默认会携带番茄种子、小麦、铁铲、草木灰和木桶。磨坊主、面包师等职业 NPC 也会携带与行业有关的原料和成品。
