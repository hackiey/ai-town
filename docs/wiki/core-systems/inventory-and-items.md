# 背包与物品

## 背包容量

| 项 | 数值 |
|---|---:|
| 背包格数 | 20 |
| 布局 | 5 x 4 |
| 默认最大堆叠 | 99 |

## 堆叠规则

两个物品只有在下列内容完全一致时才会堆叠：

- `item_id`
- 品质
- 形状
- 材质组成
- 标签
- 物品属性
- 鲜度等级

常见不能堆叠的情况：品质不同、鲜度不同、容器内容不同、材质不同。

## 物品身份

物品不是单纯的字符串 ID，而是由几层信息共同构成。

| 层 | 内容 | 用途 |
|---|---|---|
| 模板 | id、kind、最大堆叠、静态属性、行为脚本 | 定义基础物品 |
| 涌现身份 | shape、materials、tags | 制造和反应匹配 |
| 可变状态 | 容器内容、液体量、鲜度、耐久、效果 | 每个实例自己的状态 |
