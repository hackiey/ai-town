# 醉酒与生病

醉酒和生病是两个 0-100 的损伤来源。它们不会永久降低角色真实熟练度，而是在执行动作时临时降低这一次行动的表现。

## 两种损伤

| 属性 | 范围 | 增加来源 | 下降方式 |
|---|---:|---|---|
| drunk 醉酒 | 0-100 | 喝酒，例如啤酒每杯 +6 | 自然衰减，每 10 游戏分钟约 -1 |
| sickness 生病 | 0-100 | 低体力、低精力、饥饿、腐烂食物、症状 | 自然症状衰减很慢，草药可缓解 |

干活惩罚使用 `max(drunk, sickness)`，不会双重叠加。

## 动作影响

| 动作 | 影响 |
|---|---|
| 烹饪、冶炼、打铁、磨粉、煮盐、采矿 | 临时降低有效熟练度 |
| 种植、除虫 | 增加失手概率 |
| 收获 | 降低产量 |
| 浇水 | 土壤实际增加水分变少，但桶水照扣 |
| 打水 | 实际打到的水变少 |
| 说话 | 醉酒会让说出的话乱码 |
| 听别人说话 | 烂醉时听到别人说话也可能乱码 |
| NPC 移动 | 醉酒 NPC 会走路踉跄停顿 |

## 资源是否返还

多数失手会照扣资源。例如醉着种田可能消耗种子但没种好，醉着浇水会扣掉桶水但没浇进土里，醉着除虫会消耗草木灰但害虫仍在。

## 治疗

| 状态 | 当前应对 |
|---|---|
| 醉酒 | 等待自然醒酒 |
| 感冒类症状 | 薄荷艾草茶缓解咳嗽、鼻塞、发冷等 |
| 肠胃病症状 | 姜车前草汤缓解腹泻、恶心、腹痛等 |
| 伤口症状 | 金盏花药膏缓解伤口疼痛、红肿等 |
| 虚劳症状 | 缬草补剂缓解乏力、头晕、手脚发软等 |

药效不是即时清空症状，而是在一段时间内慢慢改变具体症状。同一时间只保留一剂药效，重复吃药会替换当前疗程。
