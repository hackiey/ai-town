# 液体、容器与酿酒

液体只存在于液体容器类物品里。容器槽会记录液体量、液体类型和液体品质。

## 液体容器

| 物品 | 容量 | 可堆叠 | 备注 |
|---|---:|---|---|
| 木桶 | 20 升 | 否 | 通用打水、装水、烹饪和浇水 |
| 酿酒桶 | 100 升 | 否 | 兼发酵容器 |
| 杯子 | 小 | 否 | 分装和喝酒 |

## 水井

水井是无限液体源容器，不是普通工作台。取水不减少水井库存，但打水动作会消耗时间和体力。

当前基准：打满 20 升木桶需要 3 游戏分钟和 3 点体力。少量或多桶取水按尝试升数线性计算，即每 1 升约 9 游戏秒和 0.15 体力。

醉酒或生病时，时间和体力仍按尝试升数计算，但实际进桶水量会按状态惩罚变少。

操作流程：

1. 靠近水井。
2. 鼠标悬停水井，按 `E`。
3. 在取水面板选择背包里的可装水容器和数量。
4. 确认取水，等待动作完成后水才会进入容器。

## 倒液体

| 操作 | 说明 |
|---|---|
| 右键液体容器 | 打开倒液体分离面板 |
| 拖进工作台 | 默认倒入全部液体 |
| 右键投料 | 可选择倒入几升 |
| 工作台未消耗液体 | 关面板或取回时按原桶地址倒回 |

容器间互倒要求液体类型相同，或目标容器为空。品质按量加权平均。

## 按份取出饮品

部分饮品物品可以声明 `serving_liters`，表示 1 份饮品对应多少升桶装液体。

当前麦芽酒 `beer` 的 `serving_liters = 0.5`。从酒桶里的 `beer` 液体取到背包或货架时，系统会消耗对应升数并生成离散 `beer` 物品，品质继承酒桶液体品质。生成的 `beer` 可放在背包里并用 `use_item` 直接喝掉。

这只是零售和随身携带抽象：酿酒桶、木桶和其它液体容器仍负责生产、储存、打水、烹饪和浇田等批量液体操作。

## 水的主要消耗

| 用途 | 消耗 |
|---|---|
| 和面 | 面粉 + 水 |
| 果酱和炖菜 | 食材 + 水 |
| 浇水 | 整片田一次消耗 20 升 |
| 酿酒 | 酿酒桶灌水后按比例加入麦芽 |

## 酿酒

当前酿酒流程：

1. 用水井把酿酒桶灌满水。
2. 准备麦芽，比例为 1 麦芽 : 1 升水。
3. 右键酿酒桶，选择“酿酒”。
4. 桶里的水立刻变成品质 0 的啤酒。
5. 之后 48 游戏小时内，品质线性爬到上限。

品质上限由麦芽品质、酿酒熟练度和配方难度共同决定。

## 被动转化

晾晒和发酵都是被动转化。开始后物品先变成目标物，品质从 0 慢慢爬到上限。

| 转化 | 规则 |
|---|---|
| 小麦晾晒成麦芽 | 24 游戏小时到顶，品质上限等于小麦品质 |
| 水发酵成啤酒 | 48 游戏小时到顶，品质上限受麦芽和熟练度影响 |
