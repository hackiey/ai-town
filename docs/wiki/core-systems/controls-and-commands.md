# 操作与指令

## 基本操作

| 操作 | 说明 |
|---|---|
| `E` | 与工作站、农田、水井、货架等对象交互 |
| `B` | 打开或关闭背包 |
| `ESC` | 关闭当前面板 |
| 背包右键 | 打开使用、丢弃或分离相关菜单 |
| 拖拽 | 背包内换位，或把物品拖进工作台、农田、容器 |
| 普通输入 | 对附近 NPC 说话 |
| `/command ...` | 把自然语言命令发给后台代理 |

多个可交互对象挨在一起时，用鼠标悬停指定要操作的对象，再按 `E`。

## Slash 指令

| 指令 | 作用 |
|---|---|
| `/eat <slot>` | 吃掉背包某一格里的食物 |
| `/plant <item_id>` | 向面前空农田种下一份可种植作物 |
| `/water` | 给面前作物浇水 |
| `/pest` | 给面前作物除虫 |
| `/harvest` | 收获面前成熟作物 |
| `/farmtest [seed_id]` | 测试用批量种植 |
| `/timewarp <倍率>` | 改变时间倍率，偏调试用途 |
| `/give ...` | 调试发物品 |

## 重要提醒

- 食物恢复数值要用 `/eat <slot>`。
- 当前背包右键“使用”会消耗 1 个物品，但不等于吃东西。
- `/command` 适合说目标，不适合精确替你拼复杂配方。
