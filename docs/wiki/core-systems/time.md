# 时间系统

游戏时间持续流动，并且明显快于现实时间。

## 时间倍率

| 换算 | 数值 |
|---|---|
| 1 现实秒 | 7 游戏秒 |
| 1 现实分钟 | 7 游戏分钟 |
| 1 游戏小时 | 约 8 分 34 秒现实时间 |
| 1 游戏天 | 约 3 小时 25 分 43 秒现实时间 |
| 1 游戏年 | 360 游戏天 |

## 小时推进

每过 1 个游戏小时，世界会统一推进一次：

- 作物生长。
- 作物水分下降。
- 作物照料评分结算。
- 农田虫害判定。
- 可腐烂物品老化。
- 角色慢速状态结算。

## 时间影响最大的地方

| 系统 | 影响 |
|---|---|
| 制作 | 有持续时间，按游戏秒计 |
| 农事 | 种植、浇水、除虫、收获都有动作时间 |
| 作物 | 小麦和亚麻约 72 游戏小时完整周期，番茄约 80 游戏小时 |
| 食物 | 带保鲜配置的食物会随时间降鲜并腐烂 |
| 角色状态 | 饱食、精力、体力恢复和醉酒/生病衰减会随 tick 推进 |
