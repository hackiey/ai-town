# 品质

品质是 1-100 的整数，影响食物恢复量、作物收获量和制作结果品质。

## 品质等级

| 品质区间 | 等级 |
|---|---|
| 90-100 | 极品 |
| 70-89 | 优 |
| 40-69 | 普通 |
| 1-39 | 次品 |

## 品质倍率

品质倍率是线性的。

| 品质 | 倍率 |
|---:|---:|
| 100 | 1.00x |
| 70 | 0.70x |
| 40 | 0.40x |
| 1 | 0.01x |

## 影响范围

| 系统 | 影响 |
|---|---|
| 食物 | 恢复量 = 基础恢复量 x 品质倍率 |
| 作物 | 收获量 = 基础数量 x 品质倍率，四舍五入，至少 1 |
| 制作 | 产物品质受输入品质、手艺和批量投入惩罚影响 |
| 堆叠 | 品质不同通常不能堆叠 |

## 批量投入惩罚

同一种输入一次塞太多，产物品质会下降。

| 某一输入投入份数 | 品质乘数 |
|---:|---:|
| 1 | 1.00 |
| 2 | 0.70 |
| 3 | 0.40 |
| 4 | 0.15 |
| 5 及以上 | 0.00 |

多放材料不一定更赚。很多配方最稳的做法仍然是按标准份数做。
