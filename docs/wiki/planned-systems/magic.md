# 魔法

魔法系统当前属于规划内容，不应写入当前可玩规则。

## 核心设想

玩家可以用自然语言描述物品或法术，由 LLM 生成沙箱化 Lua，让它成为世界中可用的对象或行为。

```text
玩家描述
-> LLM 生成沙箱代码
-> 验证并注册
-> 使用时执行
-> 按资源代价结算
```

## 知识系统

知识被设计成世界里的物，而不是单纯技能点。

| 获取方式 | 说明 |
|---|---|
| 法术书/技术书 | 遗迹、商人、任务、NPC 持有 |
| NPC 师承 | 建立关系后学习 |
| 逆向工程 | 拆解他人创造物，消耗时间和资源 |

脑内知识学会后不能被夺走，但书可以被偷、抢、复制、出售。
