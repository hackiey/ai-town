# 战斗

战斗系统当前是设计稿，尚不是当前核心可玩系统。

## 设计方向

目标是哈利波特式魔杖对战：

- 真 3D 弹道。
- 可见咒语飞行。
- 躺地能躲。
- 柱子和掩体能挡。
- 魔杖储能作为施法资源。
- 体力用于奔跑、闪避和受击表现。

## 时间分层

| 层 | 频率 | 责任 |
|---|---|---|
| Frame | 60 Hz | 弹道、动画、玩家输入、视觉特效 |
| Fast tick | 4 Hz | 行为树评估、buff/status 倒计时、命中结算 |
| Hour tick | 现有慢 tick | 长期恢复、仪式、衰减等 |
| LLM async | 异步 | 战前/战后记忆和长期策略，战斗中暂停 |

## 核心边界

- 战斗每帧或每 tick 不进 LLM。
- NPC 即时战斗使用行为树。
- 咒语分投递层和物理反应层。
- 角色没有 mana，施法消耗魔杖储能。
