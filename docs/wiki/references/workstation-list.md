# 工作站清单

## 玩家常用工作站

| 工作站 | 主要动词 | 主要产物 |
|---|---|---|
| workbench 工作台 | carve、combine、mix | 木杆、木板、绳、工具、面团 |
| forge 熔炉 | fire | 金属锭、青铜锭 |
| anvil 铁砧 | shape | 金属刃、镐头、斧头 |
| mill 磨坊 | grind | 面粉 |
| drying_rack 晾晒架 | dry、passive | 种子、纤维、麦芽 |
| stove 灶台 | bake、mix | 面包、肉、蛋饼、果酱、炖菜 |
| well 水井 | 取水面板，按升数耗时 | 水 |
| saltworks_pan 盐锅 | boil | 盐 |
