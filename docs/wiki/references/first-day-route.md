# 第一天路线

这是一条当前版本里比较稳定的开局路线，用来体验水、容器、食物链、工具链和农场。

## 推荐路线

1. 去水井，把起始木桶装满水（20 升需要 3 游戏分钟和 3 体力）。
2. 到磨坊，把一部分小麦磨成面粉。
3. 在工作台，用面粉和水和成面团。
4. 到灶台，把面团烤成面包。
5. 保留一部分小麦作播种粮。
6. 有余力时走铁矿石 -> 铁锭 -> 铁刃 -> 木杆/绳 -> 铁铲路线。
7. 带番茄种子或小麦去农田种第一轮作物。

## 为什么先打水

开局有木桶但没有水。水会用于和面、炖煮、果酱、浇田和酿酒，是早期最常用的基础资源之一。打水按尝试升数消耗时间和体力，醉酒或生病时实际进桶水量会变少。

## 为什么做面包

面包恢复饱食高，保鲜时间也长，是当前最可靠的早期食物之一。

## 为什么做铁铲

铁铲是铲除作物所需工具，也能带你体验熔炉、铁砧、工作台、绳和木杆的基础工具链。
