# 已知限制

## 当前玩法限制

| 内容 | 现状 |
|---|---|
| 吃东西 | 必须使用 `/eat <slot>`，背包右键“使用”不触发食物效果 |
| 普通制作体力 | 当前不实际扣体力 |
| 普通制作失败率 | 当前现有配方实际失败率为 0% |
| 镰刀 | 能做出来，但当前收获流程不强制要求 |
| 番茄配方 | 番茄可直接吃和留种，暂未接入炖菜/果酱链 |
| 魔法 | 规划中，当前不是核心可玩系统 |
| 战斗 | 设计稿，当前不是核心可玩系统 |
| 完整任务树 | NPC 当前不是完整任务树 NPC |

## 文档口径差异

`game-mechanics.md` 记录当前可玩机制时写的是 25 名 NPC。`town-npcs.md` 记录的数据和设定已经扩展到约 50 名 NPC。wiki 中涉及“当前可玩”时优先使用前者，涉及城镇职业分工时参考后者。

## 调试指令

调试指令可能破坏正常经济节奏，不建议在正式游玩说明里作为常规玩法推荐。

| 指令 | 用途 |
|---|---|
| `/give ...` | 直接发物品 |
| `/timewarp <倍率>` | 改变时间倍率 |
| `/farmtest [seed_id]` | 测试批量种植 |
