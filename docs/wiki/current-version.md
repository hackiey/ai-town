# 当前版本

当前版本的核心玩法是一个可互动的中世纪 AI 小镇原型。玩家可以移动、说话、管理背包、制造工具和食物、种植作物，并用 `/command` 给角色下达自然语言指令。

## 当前可玩循环

1. 在小镇里移动，靠近 NPC、工作站、农田、水井或货架。
2. 打开背包，整理材料、食物、工具和容器。
3. 在工作站投料，把原料加工成中间件、工具和食物。
4. 在农田播种、浇水、除虫、收获。
5. 和附近 NPC 说话，或用 `/command` 让后台代理解析高层指令。
6. 关注饱食、精力和体力，避免饿到体力上限下降。

## 最值得记住的规则

1. 新角色有 1 个木桶，但桶里没有水，通常第一件事是去水井打水。
2. 食物真正生效的吃法是 `/eat <slot>`，背包右键“使用”目前不会自动触发食物效果。
3. 饱食会自然下降，体力上限和恢复会受饱食、精力影响。
4. 普通制作当前不会实际扣体力，打水、农事和采矿会扣。
5. 当前现有工作台配方实际失败率为 0%。
6. 打水按尝试升数线性耗时和扣体力，20 升 = 3 游戏分钟 + 3 体力。
7. 浇水是整片田一次扣 20 升桶水和 10 点体力。
8. 除虫要消耗草木灰，铲除要带铁铲并消耗耐久。
9. 食物会腐烂，浆果酱、盐渍菜汤、咸蛋饼和面包比较耐放。
