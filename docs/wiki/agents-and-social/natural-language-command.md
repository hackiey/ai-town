# 自然语言指令

`/command` 会把玩家输入的自然语言发给后台代理，再让角色按意图去移动、交互或执行动作。

## 使用方式

```text
/command 前往铁匠铺
/command 去最近的农田
/command 去磨坊做面粉
```

## 适合的指令

| 类型 | 示例 |
|---|---|
| 移动 | 去铁匠铺、去水井、去最近的农田 |
| 简单生产 | 去磨坊做面粉、去水井装满木桶 |
| 目标导向 | 做一些面包、把小麦磨成面粉 |

## 不适合的指令

- 极其精确的 UI 操作替代。
- 复杂多步骤且没有材料的制作计划。
- 需要当前尚未实装系统的行为，例如施法或战斗。

## 当前定位

自然语言指令是高层意图入口，不是作弊命令。它应尽量走角色能做到的移动、交互和动作。
