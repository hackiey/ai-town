# 工作站

工作站是当前生产链的核心入口。玩家靠近后悬停目标，按 `E` 打开对应面板。

## 当前可用工作站

| 工作站 | 数量 | 作用 |
|---|---:|---|
| 工作台 | 2 | 削木、搓绳、组装工具、和面 |
| 熔炉 | 1 | 冶炼矿石、合金 |
| 铁砧 | 1 | 把金属锭打成部件 |
| 磨坊 | 2 | 把谷物磨成粉 |
| 灶台 | 2 | 烘烤、混合食材 |
| 水井 | 3 | 无限水源容器，不是普通工作台；取水按升数耗时和扣体力 |
| 金矿矿井 | 1 | 持续采金矿，需要铁镐 |
| 银矿矿井 | 1 | 持续采银矿，需要铁镐 |
| 伐木台 | 1 | 通过 `chop_wood` 砍取木材，需要斧头 |
| 盐锅 | 1 | 把燃料熬成盐 |

代码和数据中还存在更多工作站资源，例如炼金台、炭窑、铁矿、铸币点和国库等。部分属于扩展或行业模拟内容。

## 输入槽位

| 工作站 | 输入槽位数 |
|---|---:|
| 工作台 | 5 |
| 熔炉 | 5 |
| 铁砧 | 5 |
| 灶台 | 5 |
| 磨坊 | 1 |
| 盐锅 | 1 |
| 水井 | 使用取水面板，完成取水动作后入桶 |

## 工作站不是配方格

当前系统不要求把固定 item_id 摆成特定形状。玩家把物品放入 staging 槽后，点击动词按钮，系统按物品属性束、工作站和动词去匹配反应。
