# 熟练度

熟练度是角色在不同手艺上的 0-100 数值。它影响工作台反应的失败率、产物品质，以及熟练度自身成长。

## 核心变量

| 符号 | 含义 | 范围 |
|---|---|---|
| `p` | 某技能熟练度 | 0-100 |
| `d` | 当前反应难度 | 0-100 |
| `q` | 单次产出品质 | 0-100 |

## 技能轴

| skill_id | 中文 | 覆盖内容 |
|---|---|---|
| mining | 挖矿 | 铁矿、银矿、金矿 |
| woodworking | 木工 | 砍木、削木板、削木杆 |
| charcoal_making | 烧炭 | 炭窑烧炭 |
| smelting | 冶炼 | 冶炼、合金、造币 |
| smithing | 锻造 | 打刃、斧头、镐头 |
| assembly | 装配 | 绳、刀、镰、斧、镐、铲 |
| cooking | 烹饪 | 和面、烤制、炖煮、腌制 |
| milling | 磨坊 | 磨粉 |
| salt_making | 制盐 | 熬盐 |
| brewing | 酿酒 | 发酵啤酒品质上限 |

## 当前可玩口径

当前已有公式和数据归属文档，但普通制作在当前版本里实际失败率仍为 0%。醉酒和生病会在执行动作时临时压低有效熟练度。

## 设计意图

| 场景 | 结果 |
|---|---|
| 学徒做基础活 | 能成长，但品质容易受限 |
| 老手做低难日常 | 稳，但成长很慢 |
| 挑战高难反应 | 失败风险高，成功时更容易突破 |
| 大师用烂材料 | 仍被材料品质卡住 |
