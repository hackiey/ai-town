# 制造系统

当前制造系统由“工作站 + 动词按钮 + 投料槽”组成。NPC 不走 UI，但会调用同一张反应表，因此玩家和 NPC 的产出逻辑一致。

## 基本流程

1. 靠近工作站，悬停目标并按 `E`。
2. 从背包拖物品到工作站输入槽。
3. 选择动词或具体子选项。
4. 系统匹配反应表。
5. 有持续时间的制作会显示进度条。
6. 完成后扣除被消耗的输入，把输出放回背包。

## 投料与取回

| 操作 | 规则 |
|---|---|
| 拖拽 | 全量移动或倒入全部液体 |
| 右键 | 打开分离面板，选择数量或液体升数 |
| 关面板 | 未消耗物品原路退回 |
| 取消制作 | 未消耗物品原路退回 |
| 放不回去 | 报错并留在工作台，绝不丢弃 |

液体会精确倒回原桶，避免不同品质或不同内容的液体被混到一起。

## 制作时间

所有制作时间按游戏秒计时，会随时间倍率加速。

| 游戏秒 | 约现实时间 |
|---:|---:|
| 70 | 10 秒 |
| 140 | 20 秒 |
| 210 | 30 秒 |
| 420 | 60 秒 |

## 当前失败率

当前现有配方虽然有难度和失败文案，但玩家 mastery 固定为 `1.0`，配方 difficulty 不超过 `0.3`，所以实际失败率为 0%。

## 当前制作成本

- 时间。
- 原材料。
- 容器里的水。
- 打水、采矿和农事会消耗体力。
- 普通制作当前不会实际扣玩家体力。

## 批量投入

同一输入投入太多会降低产物品质。见 [品质](../core-systems/quality.md)。

## 示例路线

| 路线 | 步骤 |
|---|---|
| 铁锹 | 铁矿石 + 木炭 -> 铁锭 -> 铁刃；木材 -> 木杆；纤维 -> 绳；铁刃 + 木杆 + 绳 -> 铁铲 |
| 面包 | 小麦 -> 面粉；面粉 + 水 -> 面团；面团 -> 面包 |
| 腌肉 | 生肉 + 盐 -> 腌肉 |
