# 采矿、伐木与制盐

这些系统提供生产链上游资源。当前经济设计强调资源来自节点和劳动，而不是商人凭空刷新。

## 采矿

| 矿点 | 产物 | 要求 |
|---|---|---|
| 铁矿 | 铁矿石 | 工具链上游，文档中由铁矿把头负责 |
| 银矿 | 银矿石 | 需要铁镐 |
| 金矿 | 金矿石 | 需要铁镐 |

采矿一次工具调用覆盖 1 游戏小时。Godot server 内部每 5 游戏分钟进行一次产出判定，每次先消耗 10 点体力，体力不足会提前结束。

## 伐木与木炭

| 环节 | 说明 |
|---|---|
| 伐木场 | 通过 `chop_wood` 提供 wood，服务炭窑和木杆链 |
| 炭窑 | 把 wood 烧成 charcoal |
| 木炭 | 当前主要燃料，用于熔炉和盐锅 |

经济模型中，木材如果没有足够建筑和维修消耗，会优先通过炭窑进入燃料链。

## 制盐

| 工作站 | 输入 | 输出 |
|---|---|---|
| 盐锅 | 燃料 x1 | 盐 x20 |

盐用于腌肉、咸蛋饼和盐渍菜汤，是保藏食品链的重要材料。

## 资源获取路径

玩家通常有三种供给路径：

| 路径 | 成本 |
|---|---|
| 自己采 | 时间、工具、体力 |
| 市集买 | 钱，受 NPC 产能和营业时间影响 |
| 自己加工 | 时间、工作站、上游材料 |
