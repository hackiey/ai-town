# 配方速查

本页记录当前玩家可见的主要工作站配方。时间按游戏秒计算。

## 工作台

| 动作 | 输入 | 输出 | 游戏秒 |
|---|---|---|---:|
| 削木杆 | 木材 x1 | 木杆 x1 | 140 |
| 削木板 | 木材 x1 | 木板 x1 | 140 |
| 搓绳 | 纤维 x1 | 绳 x1 | 70 |
| 组装铁刀 | 铁刃 x1 + 木杆 x1 | 铁刀 x1 | 70 |
| 组装镰刀 | 铁刃 x1 + 木杆 x1 | 镰刀 x1 | 84 |
| 组装铁锹 | 铁刃 x1 + 木杆 x1 + 绳 x1 | 铁铲 x1 | 105 |
| 组装铁镐 | 铁镐头 x1 + 木杆 x1 + 绳 x1 | 铁镐 x1 | 105 |
| 组装铁斧 | 铁斧头 x1 + 木杆 x1 + 绳 x1 | 铁斧 x1 | 105 |
| 和面 | 面粉 x1 + 水 x1 | 面团 x1 | 70 |
